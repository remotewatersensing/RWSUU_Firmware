void calibConductivity ( void  )

Calibrates the conductivity sensor Blocking function which takes control of the serial interface, printing out the temperature and ms/cm repeatedly. Calibrates and saves the values on the EEPROM. To calibrate, connect the MCU with a serial monitor. The following steps should be taken in sequence:

  1. Make sure the sensor is not submerged
  2. Send: enterec
  3. Put your sensor in a 12.88ms/cm solution
  4. Send: calec
  5. If it returns a success, send: exitec If it fails, try again
    Returns
    float
